# RFM-анализ клиентской базы аптечной сети

## Цели проведения анализа клиентской базы:
Распределить клиентов по группам и сформировать рекомендации для каждой группы клиентов.

## Для проведения RFM-анализа используются классические метрики:
- **Recency (давность)**: определение даты последней покупки каждого клиента. 
- **Frequency (частота)**: подсчет количества покупок каждого клиента за определенный период. 
- **Monetary (сумма)**: расчет общей суммы, потраченной каждым клиентом за этот период. 

## Используемые инструменты в работе:
1. **SQL** - группировки GROUP BY, агрегирующие функции (SUM, COUNT, MIN), перцентили (PERCENTILE_CONT), CTE, распределение клиентов по группам с помощью оператора CASE, CROSS JOIN
2. **Metabase** - для отображения основных метрик и построения дашборда
